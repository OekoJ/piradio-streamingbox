Die Idee der Piradio Streamingbox
---------------------------------
Die Piradio Streamingbox soll ein Audiosignal in einer Qualität, die für Radiosendungen ausreichend ist, von einem Ort zum anderen übertragen. Sie dient also zur Live-Übertragung von Radiosendungen, kann aber natürlich auch als Live-Stream für Veranstaltungen oder für das Übertragen von Vogelgezwitscher aus dem Baumhaus (mit Internetanschluss) genutzt werden.
